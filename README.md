A Final Year Project developed using **MongoDB, Express.js, React.js, and Node.js** with AI integration through OpenRouter API.

Abstract
The AI ChatBot is a web-based intelligent chatbot application designed to provide users with a secure and interactive conversational experience. The system enables users to register, authenticate, communicate with an AI assistant, upload PDF documents for AI-assisted interaction, and analyze resumes. Built using the MERN Stack, the application delivers a responsive interface and secure backend architecture while integrating modern AI capabilities.
---
Objectives
- Develop an AI-powered chatbot using the MERN Stack.
- Implement secure user authentication using JWT.
- Enable AI-based conversational interaction.
- Allow users to upload PDF documents and receive AI-generated responses.
- Provide resume analysis functionality.
- Deliver a responsive and user-friendly interface.

Installation
  Clone Repository
```bash
git clone https://github.com/Ramyas14/AIChatBot.git
```
Install Dependencies
 Client
```bash
cd client
npm install
```
  Server

```bash
cd server
npm install
---
 Environment Variables
Create a `.env` file inside the **server** folder.

```env
PORT=5000
MONGODB_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
OPENROUTER_API_KEY=YOUR_API_KEY
---
 Run the Project
Backend
```bash
cd server
npm run dev
```
 Frontend
```bash
cd client
npm run dev
```

```
http://localhost:5173
